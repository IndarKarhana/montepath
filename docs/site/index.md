# montepath Docs

`montepath` is an agent-native Monte Carlo runtime with a Python-first user
surface and a compiled Rust execution core.

Start here:

- `quickstart.md`: install and price the first option.
- `../public-alpha.md`: public alpha positioning, limitations, and upcoming features.
- `../uv-and-agent-install.md`: install with `uv` and run MCP tools through `uvx`.
- `api-reference.md`: current Python public API.
- `benchmark-interpretation.md`: how to read timing and accuracy claims.
- `agent-tools.md`: stable agent-safe request/response wrappers.
- `migration-notes.md`: what is stable today and what may change before `1.0`.
